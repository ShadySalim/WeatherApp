# WeatherApp
A lightweight weather search application built with HTML, CSS, and Vanilla JavaScript, using the OpenWeather API. Users can search for any city and instantly view key weather information such as temperature, humidity, wind speed, country name, and current sky conditions with live API icons.


Features
	•	Search weather by city name
	•	Clean UI with real-time weather updates
	•	Handles invalid city searches with a custom error message
	•	Displays:
	•	Temperature (°C)
	•	Humidity (%)
	•	Wind speed
	•	City & country
	•	Sky condition description
	•	Official OpenWeather sky icon
